
\section{Motivation and Overview}
\label{subsec:parallel_motivation}
\hspace{\parindent}
Since TEA* is sensitive to the selected priority ordering and the most suitable heuristic is scenario-dependent, the complementary heuristics introduced in Chapter~5 motivate the evaluation of multiple ordering strategies rather than relying on a single fixed priority rule.

A direct way of evaluating several heuristics would be to run TEA* multiple times sequentially,
each time using a different priority ordering. However, this would increase the planning latency
approximately in proportion to the number of evaluated heuristics, which is undesirable in an
online coordination system. To address this limitation, the proposed framework evaluates multiple
priority orderings in parallel. Each worker thread runs an independent TEA* instance using an
ordering generated by a different heuristic, and the resulting candidate plans are compared after
the parallel execution stage.

This approach preserves TEA* as the core planning algorithm, while extending the surrounding
execution framework to exploit parallel computation. The objective is to improve solution quality by evaluating several priority orderings within a planning time
close to that of a single TEA* execution.
